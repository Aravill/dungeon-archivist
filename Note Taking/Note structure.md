- Have a directory structure like this:
```
Lore
	|- Characters
	|- Events
	|- Locations
	|- Items
Campaigns
	|- Court of Dusk and Dawn
		|- Characters
		|- Events
		|- Locations
		|- Items
		|- Sessions
			|- Session 1: Into the Wilder
```
- `Lore`
	- Notes relevant for the whole world / setting and are used in several adventures / campaigns
	- This is where your world building happens
- `Campaigns` 
	- Notes relevant for specific adventures / campaigns
	- Here is where you will also find session notes
- If you're using Obsidian (you should), You can then create links across your note files
