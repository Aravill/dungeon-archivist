# How to prepare
- Figure out how the session ties into your campaign or adventure
	- If it's a one-shot, you don't need to think about this continuity
- Use the [[Session Notes Template]] file
	- The template defines the structure of how I prefer to keep my notes
- Fill out its sections using the checklist below
# Checklist
- [ ] Created a new session note file
- [ ] Defined session goals
- [ ] Read the previous session notes
- [ ] Created a "previously"
- [ ] Noted down do-not-forgets
- [ ] Scenes were set up
	- [ ] Each scene has a purpose, backdrop & links to stat blocks if necessary
